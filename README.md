# mittere

Mittere is a private and a non-censoring chat application with a server for anyone to run. The server is security driven, with features like no connection to any other servers (standalone), and an option to only allow users with a specially generated key to join (keys are generated by the specific server). 
No data is ever saved or sent anywhere other than to the server and data such as display names and messages are the only things sent to clients about other clients. The server will not store any data other than what is needed to function, such as login data, Channel message logs (see below), and connected users.
